## 🚀 Configuración Inicial

### **1. Clonar el Repositorio Maestro**
```bash
git clone https://github.com/https-panacea-icono-org/MASTER-REPOSITORY-ELEUSIS-ICONO.git
cd MASTER-REPOSITORY-ELEUSIS-ICONO
```

### **2. Inicializar Submodules**
```bash
# Inicializar todos los submodules
git submodule init
git submodule update --recursive

# O en un solo comando
git submodule update --init --recursive
```

### **3. Configurar Repositorios Remotos**
```bash
# Ejecutar script de configuración
./scripts/setup-repositories.sh
```

## 🔄 Sincronización de Repositorios

### **Sincronización Automática**
```bash
# Sincronizar todos los repositorios
./scripts/sync-repositories.sh

# Sincronizar un repositorio específico
./scripts/sync-repositories.sh panas-app-interface
```

### **Actualización de Submodules**
```bash
# Actualizar todos los submodules
./scripts/update-submodules.sh

# Actualizar un submodule específico
git submodule update --remote core-projects/panas-app-interface
```
